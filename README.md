# Decal Monkey Field App

Decal Monkey Field App is a live internal Printavo-connected field workflow for sales, measurements, installer closeout, customer history, and shared app logins.

## Repository Safety

This repository intentionally excludes live server data, uploads, built frontend bundles, backups, dependency folders, and local environment files. Keep real Printavo credentials, inquiry keys, app usernames, app passwords, and session data out of source control.

Use `Server/.env.example` as the template for local or deployed server configuration.
